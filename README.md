# Sherlock Holmes

OSINT engine designed for tracking identities across multiple platforms.

## Structure

- `packages/sdk`: Core types, base classes, and scoring engine.
- `packages/core`: Central engine, orchestration, and infrastructure implementations.
- `packages/cli`: Command-line interface.
- `packages/providers`: Built-in provider implementations.

## Development

This project uses [Bun](https://bun.sh) and [Biome](https://biomejs.dev).

```bash
bun install
bun test
bun run lint
```
